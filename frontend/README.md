## 📖 About

Bellezaa is a modern AI-powered beauty and skincare platform that helps users build personalized skincare routines, discover suitable products, and receive instant beauty advice through an intelligent AI assistant.

Instead of generic recommendations, Bellezaa focuses on personalization by understanding the user's skin type, concerns, and goals.

The project combines modern frontend technologies with AI to provide a smooth, interactive, and premium user experience.

# 🚀 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/bellezaa.git
```

```
cd bellezaa
```

---

## Install Frontend

```bash
cd frontend
npm install
```

---

## Install Backend

```bash
cd backend
npm install
```

---

# ▶ Run Frontend

```bash
npm run dev
```

Runs on

```
http://localhost:5173
```

---

# ▶ Run Backend

```bash
npm run dev
```

Runs on

```
http://localhost:8000
```

---

# 🔑 Environment Variables

Create a `.env` file inside backend.

```env
PORT=8000

MONGODB_URI=your_mongodb_uri

JWT_SECRET=your_secret_key

GROQ_API_KEY=your_groq_api_key
```
